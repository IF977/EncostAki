<h1> Histórias de Usuário</h1>


```sh

Como usuário cadastrado

Quero poder informar os chamados que abri

Para que todos os outros usuários possam visualizar e acompanhar junto comigo

```

```sh

Como usuário cadastrado

Quero poder postar fotos do estado em que se encontra a barreira/encosta próxima de onde resido

Para que o problema tenha visibilidade e possa chamar atenção das autoridades competentes

```

```sh
Como usuário cadastrado

Quero poder comentar em fotos e localizações que foram colocadas

Para que eu possa interagir com a comunidade quanto aos problemas daquele local

```

```sh
Como usuário visitante

Quero ter acesso a informações das barreiras/encostas da região em que resido

Para que tanto eu como a comunidade saiba quais e quantos chamados devem ser abertos para resolver meu problema
```
